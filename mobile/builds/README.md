# Сборка мобильного приложения (iOS и Android)

В этой папке собраны инструкции и скрипты для компиляции и сборки вашего мобильного приложения.

Так как вы разрабатываете на **Windows**, локально скомпилировать `.ipa` для iOS через Xcode невозможно. Поэтому мы настроили **EAS Build** (облачный компилятор Expo), который выполняет всю сборку на серверах Apple (macOS) в облаке.

---

## 🛠 Подготовка к сборке

Перед запуском любого скрипта сборки убедитесь, что:
1. Установлен глобальный CLI-пакет EAS:
   ```bash
   npm install -g eas-cli
   ```
2. Вы вошли в свой Expo-аккаунт:
   ```bash
   eas login
   ```

---

## 🚀 Скрипты для сборки

Для вашего удобства в этой папке созданы готовые скрипты автоматизации:

### 1. Сборка для Симулятора iOS (Без Apple Developer Account)
Этот скрипт скомпилирует приложение под симулятор iOS. Идеально подходит для проверки работы на эмуляторе Mac или передачи тестировщикам.
* **Скрипт:** `build-preview-ios.ps1`
* **Команда (запуск из этой папки):**
  ```powershell
  ./build-preview-ios.ps1
  ```

### 2. Сборка APK для Android (Для установки на любой Android-телефон)
Этот скрипт запускает сборку полноценного **`.apk`** файла, который можно сразу скинуть на любой Android-телефон и установить напрямую (без Google Play).
* **Скрипт:** `build-apk-android.ps1`
* **Команда (запуск из этой папки):**
  ```powershell
  ./build-apk-android.ps1
  ```

### 3. Сборка для реального устройства iOS (Ad-Hoc / App Store)
Скрипт запускает сборку полноценного `.ipa` файла для реального iPhone. Требует наличия Apple Developer Credentials.
* **Скрипт:** `build-production-ios.ps1`
* **Команда:**
  ```powershell
  ./build-production-ios.ps1
  ```

### 4. Предварительная локальная генерация нативных проектов (Prebuild)
Если вы хотите локально сгенерировать папки `ios/` и `android/` с нативным кодом (для инспекции или ручного редактирования):
* **Скрипт:** `run-prebuild.ps1`
* **Команда:**
  ```powershell
  ./run-prebuild.ps1
  ```

## 📂 Файлы конфигурации

* **`eas.json`** (в корне папки `mobile/`) — содержит профили сборки (`development`, `preview`, `production`).
* **`app.json`** (в корне папки `mobile/`) — настроен с уникальным идентификатором пакета: `de.empirepremiumbau.admin` для iOS и Android.
